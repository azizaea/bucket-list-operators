# Bucket List - Operators

## Overview
Saudi Arabia's first AI-powered tour operator management platform.

Tour operators can:
- Manage tours and schedules
- Accept bookings with real-time availability
- Process payments (Mada, SADAD, Apple Pay)
- Generate Ministry of Tourism compliance reports
- Manage resources (guides, vehicles, equipment)

## Tech Stack
- **Backend:** Node.js + Express + TypeScript
- **Database:** PostgreSQL with Prisma ORM
- **Payments:** Moyasar (Saudi payment gateway)
- **Deployment:** Digital Ocean
